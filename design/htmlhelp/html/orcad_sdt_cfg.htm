Orcad sdt.cfg example
 

{ OrCAD/SDT IV  Configuration File }


PDRV   = 'D:\ORCADESP\DRV\'


PSCH   = ''


PLIB   = 'D:\ORCADESP\SDT\LIBRARY\'


DD     = 'VGA640.DRV'


PRD    = ''


PLD    = 'HP.DRV'


LIB    = 'TTL.LIB'


LIB    = 'CMOS.LIB'


LIB    = 'DEVICE.LIB'

If necessary such a sdt.cfg file can be created. The lines containing the strings PLIB and LIB are important, and the first line for identification.

The PLIB string will contain the library directory, and the LIB strings the individual libraries in the library directory.
