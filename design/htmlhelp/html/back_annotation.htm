Back annotation
 

Press Back annotation button

Back annotation means the changes made with the gate and pin swaps will be reflected into the schematics. When the Back annotation function is executed, the file pcb\gatepin.ban will be red, and the necessary schematics will be modified.

For example: The two pins of a 7400 ttl device have been swapped by using the layout editor. After Back annotation the wires connected to those two 7400 pins in the schematics will be switched.
