Import from DXF
 

Sub menu File menu item Import from dxf

With this function a dxf file can be imported. In the next dialogbox a filename is requested. After OK the file will scanned for available layers. After scanning these layers will be visible in a new dialogbox. After selecting the required layers and pressing OK the objects inside the dxf file will be imported. Only the objects on the selected will be imported. The objects in the dxf file will be stored as lines/circles/arcs or text. Dimensions inside the dxf file are not supported. The imported objects will be selected, and if necessary these objects can be scaled/rotated or moved.
