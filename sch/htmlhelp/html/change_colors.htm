Change colors
 

Sub menu View menu item Change colors

The color settings can be modified in the next dialogbox. The color settings will be copied into the sch.ini initialization file. This file is stored into the directory of the design.

To use those sch colors for new designs, copy this sch.ini file to the projects directory. Whenever a new design is created this sch.ini file in the projects directory will be copied to the directory of the new design.
