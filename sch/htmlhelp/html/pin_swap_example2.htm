Gate swap example 1
 

74245 TTL device with eight inputs and outputs. Every Input/Output combination can be swapped against any other seven Input/Outputs. This means there are eight gates, each with two pins.

The gate/pin swap dialog window should be filled like:

  Code Pins
Line 1 1 2,18
Line 2 1 3,17
Line 3 1 4,16
Line 4 1 5,15
Line 5 1 6,14
Line 6 1 7,13
Line 7 1 8,12
Line 8 1 9,11

All the pins in lines with the same code numbers can be swapped. This means the number of pins in each line should be the same, and the numbers of pins enclosed by parentheses should be the same.
