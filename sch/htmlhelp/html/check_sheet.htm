Check sheet
 

Sub menu Edit menu item Check

The current sheet will be checked for warnings/errors. Possible warnings/errors are :

  • Not connected busconnections.

  • Not connected external connections.

  • External connections.must be unique.

  • External connections directy connected to a busconnection.

  • Wires directly connected to a bus.

  • Busconnections not connected properly to a wire/bus.

  • In a sheet the pins of a symbol are not connected to a wire/bus endpoint.

  • Netlabel is not connected to a wire or bus.

  • Symbol pins without a one pinnet mark and without any other connection

  • Overlapping wires/busses

  • Lose wires
