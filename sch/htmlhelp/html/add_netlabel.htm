Add netlabel
 

Press Add label to wire/bus button
Press n
Add label to wire/bus

A netlabel is a text string that identifying the wire or bus. When a wire is connected to a bus via a busconnection, the wire needs a netlabel. A netlabel is always required for a bus. When wires or busses are drawn with two or more lines, only one line of them needs this netlabel.

To add a netlabel to a wire or bus, select the wire or bus first. After the wire or busses has been selected use this function. After the function is activated a dialogbox is visible. You have to fill in a name for the netlabel. After pressing theOK button, the netlabel can be placed. There is a help line visible, to show at which endpoint of the wire or bus the netlabel is connected.
