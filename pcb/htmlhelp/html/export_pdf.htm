Export to PDF
 

Sub menu File item Export item Plotfiles as PDF

Plot selected layers in the PDF format. In the next dialogbox the layers, papersize, paper orientation and plotting board outline can be selected. There are also two editboxes available. In those two editboxes (each four lines) some information about the PCB can be stored. This information will then be plotted additionally for each layer. Initial three macros are stored into the first editbox.

$DesignNameCurrent design name

$LayerCurrent layer

$DateCurrent date

The PDF file <design>.pdf will be generated in the pcb subdirectory.
