Change clearance

In the submenu of Change clearance, the clearance for the current trace can be modified. After trace drawing the new clearance width will not be active anymore. In the submenu is the current clearance width marked (If available). If none of the clearance widths is appriopiate, use the last item of the submenu. If the last menu item is selected, the clearance width can be typed in the following dialogbox.

The clearance settings in the pull down menu can be modified by changing the pcb.ini settings.
