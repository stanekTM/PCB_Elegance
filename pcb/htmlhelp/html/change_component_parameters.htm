Change component parameters
 

If the component references/values are visible, some parameters can be changed. The parameters are:

  • Edit component value text

  • Hide component reference/value text

  • Unhide component reference/value text

  • Move component reference/value text to top layer

  • Move component reference/value text to bottom layer (Reversed text)

  • Change textheight component reference/value

  • Change text line width component reference/value
