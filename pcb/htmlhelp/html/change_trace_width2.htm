Change trace width
 

Change trace width
Press w

In the submenu of Change trace width, the trace width for the current trace can be modified. After trace drawing the new trace width will not be active anymore. In the submenu is the current trace width marked (If available). If none of the trace widths is appriopiate, use the last item of the submenu. If the last menu item is selected, the trace width can be typed in the following dialogbox.

The trace settings in the pull down menu can be modified by changing the pcb.ini settings.
