Copy traces/vias from clipboard

If there are traces/vias stored on the clipboard, they will be imported and displayed. The traces/vias can now be moved to the correct position. To make this more easy, snap functionality has been included. After clicking on the left mouse button, the mouse cursor will snap to the object closest to the cursor. If necessary the moving center can be changed by holding the shift key. After release the moving center will snap to the moving object closest to the mouse cursor.
