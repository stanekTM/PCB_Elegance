Export to DXF file
 

Sub menu File menu item Export to DXF file

Exports the selected layers to a DXF file. In the next dialogbox the layers and drawing board outline can be selected. There are two options available: Filled objects means the objects will be exactly represented in the DXF file. If this option is not marked, only the hairlines will be put into the dxf file. The second option is a mirror in the X direction. After pressing the OK button the DXF file will be generated.
