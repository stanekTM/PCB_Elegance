Add text objects
 

Add text -> select layer

A text object will be added. In the next dialogbox the text can be entered. In addition the textheight can be edited. After pressing the OK button the text can be placed. When the spacebar is pressed, a dialogbox will popup, and the text placement point can be edited by hand. When the first character typed is a @ the coordinates will be relative against the Relative (grid)position. The coordinates typed in will be used with the current units (dimension).
