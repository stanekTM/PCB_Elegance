Add arc objects
 

Add arc -> select layer

An arc object will be added. When the spacebar is pressed, a dialogbox will popup, and the arc parameters can be edited by hand. The first parameters are the diameter. The optional second and third parameter is the arc center. The optional fourth and fifth parameter is the first radial ending point. The optional sixth and seventh parameter is the second radial ending point. When the first character typed is a @ the coordinates will be relative against the Relative (grid)position. The coordinates typed in will be used with the current units (dimension).
