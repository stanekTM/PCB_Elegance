Change colors
 

Sub menu View menu item Change colors

The color settings can be modified in the next dialogbox. The color settings will be copied into the geom.ini initialization file. This file is stored into the current design directory.

To use those colors for new designs, copy this geom.ini file to the projects directory. Whenever a new design is created this geom.ini file in the project directory will be copied to the directory of the new design.
