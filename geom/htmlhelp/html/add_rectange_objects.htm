Add rectangle objects
 

Press Add rectangle component outline button
Press Add rectangle placement outline button
Press r (Placement outline)
Add pad -> Rectangle -> Select layer

Add rectangle -> Select layer

A rectangle object will be added. When the spacebar is pressed, an dialogbox will popup, and the rectangle parameters can be edited by hand. The first two parameters are the width, and height. The optional third and four parameter is the rectangle center. When the first character typed is a @ the coordinates will be relative against the Relative (grid)position. The coordinates typed in will be used with the current units (dimension).
